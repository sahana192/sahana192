# Infera AI
### Intelligent Content Analysis & Summarization Platform

Infera AI is a premium, full-stack application designed to extract, clean, and summarize content from multiple formats (PDF, DOCX, URL, Text) using advanced AI models via Groq.

## 🚀 Features
- **Multi-Format Support**: Summarize PDFs, Word documents, Websites, or Raw Text.
- **Intelligent Insights**: Extract key points, sentiment analysis, and tone.
- **Interactive Chat**: Ask questions about your documents in real-time.
- **Premium Bento UI**: A modern, glassmorphic dashboard for peak productivity.
- **Functional History**: Search and manage your past analyses easily.

## 🛠️ Tech Stack
- **Frontend**: React + Vite + TailwindCSS + Lucide Icons.
- **Backend**: FastAPI + SQLAlchemy (SQLite) + Groq AI.
- **Data Persistence**: Excel-based history and notification tracking.

## 🚦 Getting Started

### Backend Setup
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
